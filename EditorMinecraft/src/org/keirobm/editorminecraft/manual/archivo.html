Menu Archivo

Opciones en el Menu Archivo

  • Nuevo: Crea un nuevo plano con 1 sola capa vacia. Pregunta antes si desea guardar el actual para no perderlo.
  • Abrir: Muestra un cuadro de dialogo para buscar y cargar un archivo de plano (con extension .mpl). Pregunta antes si desea guardar el actual para no perderlo.
  • Guardar: Guarda el plano en el archivo seleccionado con Guardar Como. Si no se ha seleccionado ningun archivo: porque no se ha dado antes a Guardar Como, esta opcion tiene el mismo comportamiento que Guardar Como.
  • Guardar Como: Muestra un cuadro de dialogo para almacenar el archivo de plano y lo guarda. Sucesivas llamadas a Guardar lo guardan en el archivo aqui seleccionado.
  • Salir Sale de la aplicacion. Pregunta antes si desea guardar el plano para no perderlo.
